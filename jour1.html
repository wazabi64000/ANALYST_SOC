2. AAA — Les 3 serrures de la maison

Analogie parfaite : une boîte de nuit VIP

  • Authentification → « C’est bien toi ? » → mot de passe, biométrie, 2FA
  • Autorisation → « Tu as le droit d’aller où ? » → bracelet VIP, lecteur de badge
  • Accounting (Audit) → « On garde la trace de tout » → caméras, logs

Exemple réel : attaque SolarWinds 2020 → les hackers ont volé un mot de passe administrateur → mauvaise authentification → ils ont eu tous les droits.

4. Les 5 fonctions NIST — Ta checklist à vie

Tu peux tout résumer avec ces 5 verbes :

  1. Identify → Qu’est-ce que j’ai à protéger ? (serveurs, données clients, etc.)
  2. Protect → Mettre des murs (firewall, antivirus, mots de passe forts)
  3. Detect → Repérer l’attaque (c’est 80 % de ton futur métier)
  4. Respond → Réagir vite (isoler la machine, bloquer l’IP)
  5. Recover → Restaurer et tirer les leçons

5. Les logs — Ton nouveau meilleur ami

Un log = une ligne dans le journal de bord du système.

Exemples concrets que tu vas voir en live aujourd’hui :

# Windows – Connexion réussie RDP

Subject: User Name: administrateur
Source Network Address: 192.168.1.55
